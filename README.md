# AI Training Hub

A comprehensive training repository for learning and practicing Generative AI concepts, patterns, and hands-on development.

## Repository Structure

```
ai-training-hub/
└── Generative AI/
    ├── 01-fundamentals/          # Core GenAI concepts and terminology
    ├── 02-llms/                  # Large Language Models deep dive
    ├── 03-prompt-engineering/    # Prompt design techniques and best practices
    ├── 04-hands-on/              # Hands-on development exercises
    ├── 05-advanced/              # RAG, fine-tuning, AI agents
    ├── 06-applications/          # Real-world use cases and projects
    ├── 07-ethics/                # Responsible AI and ethical considerations
    ├── 08-agentic-patterns/      # Agentic design patterns (8 patterns)
    ├── 09-mcp/                   # Model Context Protocol
    ├── examples/                 # Agent implementation examples
    ├── exercises/                # Practice exercises with solutions
    ├── notebooks/                # Jupyter notebooks
    ├── presentations/            # Slide decks and visual materials
    └── resources/                # Additional learning resources
```

## Getting Started

### Prerequisites

- Python 3.8+
- OpenAI API key

### Setup

```bash
# Create and activate virtual environment
python -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r "Generative AI/requirements.txt"

# Set up environment variables
# Add your OPENAI_API_KEY to Generative AI/.env
```

## Key Topics Covered

- **Fundamentals** — What is Generative AI, key concepts, model types
- **LLMs** — How large language models work, popular models overview
- **Prompt Engineering** — Basic to advanced prompting techniques
- **Hands-On Development** — Environment setup, building AI apps, API integration
- **Advanced Topics** — RAG, fine-tuning, AI agents and workflows
- **Agentic Patterns** — Single, sequential, parallel, hierarchical, and more
- **MCP** — Model Context Protocol fundamentals and use cases
- **Ethics** — Bias, fairness, responsible AI practices
